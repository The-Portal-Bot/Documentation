### Description

_returns the input capitalised, the first letter capitalcase and the rest lowercase_

### Input Type

string, string array

### Output Type

string

### Example

|          input          |   output    |
| :---------------------: | :---------: |
| __ioannis\|capitalise__ | __Ioannis__ |
