### Description

_returns the acronym of the input_

### Input Type

string, string array

### Output Type

string

### Example

|           input           |  output   |
| :-----------------------: | :-------: |
| **$status_list\|acronym** | **AL,RL** |

$status_list = [ "Apex Legends", "Apex Legends", "Rocket League" ]
