### Description

_returns the name of the most common element in list_

### Input Type

string, string array

### Output Type

string

### Example

|           input            |  output  |
| :------------------------: | :------: |
| __$status_list\|populous__ | __Apex__ |

$status_list = [ "Apex Legends", "Apex Legends", "Rocket League" ]

:::info

If there are more than one, one is selected randomly

:::
