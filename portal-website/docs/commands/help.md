## Examples

* Lets say you want to get general information, you would type
    ```bash
    ./help
    ```

* Lets say you want to learn how to use the command force, you would type
    ```bash
    ./help force
    ```

* Lets say you want to know all the commands Portal has, you would type
    ```bash
    ./help commands
    ```
