### docker-compose

1.  Go to Portal directory

        ```bash
        cd Portal
        ```

2.  Run Portal

        ```bash
        docker-compose up
        ```

3.  Stop Portal

        ```bash
        docker-compose down
        ```

---

### standalone docker container

3.  Run Portal

        ```bash
        docker run portal
        ```

4.  Stop Portal

        ```bash
        docker stop 4cfb856dc61d
        ```
