#### macOS

1.  install [deno](https://deno.com/)

        ```bash
        curl -fsSL https://deno.land/install.sh | sh
        ```

#### ubuntu

1.  install [deno](https://deno.com/)

        ```bash
        curl -fsSL https://deno.land/install.sh | sh
        ```

#### windows

1.  install [deno](https://deno.com/)

        ```bash
        irm https://deno.land/install.ps1 | iex
        ```

:::warning Make sure you have deno version ^2.0.0
