#### macOS

1.  install [homebrew](https://brew.sh)

        $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2.  update homebrew

        $ brew update

3.  install npm and node ^14.x

        $ brew install node@14
