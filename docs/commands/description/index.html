Commands

Commands are used to communicate with Portal, by typing {prefix}{command} (ex. ./about) you can access this command, and await a reply or action from Portal.

Portal prefix: / (default)

A​

about​

DescriptionArgumentClearanceCooldown
Returns an about Portal messagenoneeveryonenone

announce​

DescriptionArgumentClearanceCooldown
Makes an announcement in the announcement
channel and tags @here		!title | @bodyeveryone1min/user

announcement​

DescriptionArgumentClearanceCooldown
Creates a new announcement channel or
sets the current channel as announcement		!channel | @categoryadminnone

B​

ban​

DescriptionArgumentAccessibleCooldown
Bans a member from server@reason | @daysmoderator1min/member

bet​

DescriptionArgumentClearanceCooldown
Replies with the latest betting stats!provider !gameeveryonenone

C​

corona​

DescriptionArgumentClearanceCooldown
Replies with the daily state of corona virus cases!country_codeeveryonenone

crypto​

DescriptionArgumentClearanceCooldown
Replies with the current price of requested cryptocurrency!cryptocurrency | exchangeeveryonenone

D​

delete​

DescriptionArgumentClearanceCooldown
Deletes n number of messsages on the text channel!numer_of_messagesadminnone

F​

focus​

DescriptionArgumentClearanceCooldown
Creates focus channel for you and your requested member!username @timeadminnone

force​

DescriptionArgumentClearanceCooldown
Clones current channel in order to force-update namenoneadmin2min/admin

H​

help​

DescriptionArgumentClearanceCooldown
Returns a help-list if specified returns specific descriptionvariables/attributes/pipes/structures,
@command/@vrbl/@attr/@pipe/@struct		everyonenone

I​

ignore​

DescriptionArgumentClearanceCooldown
Ignore channels, so that portal does not listen to themnoneadminnone

invite​

DescriptionArgumentAccessibleCooldown
creates an invite link and sends it to you as a private message!role_jsonadmin1min/member

J​

join​

DescriptionArgumentClearanceCooldown
Create ignore channel or sets as ignore channel or ignore a membernoneadminnone

joke​

DescriptionArgumentClearanceCooldown
Replies with a joke@categoryeveryonenone

K​

kick​

L​

leaderboard​

DescriptionArgumentClearanceCooldown
Replies with the top 10 leaderboard@number_of_memberseveryonenone

leave​

DescriptionArgumentClearanceCooldown
Removes Portal from voice channelnoneeveryonenone

level​

DescriptionArgumentClearanceCooldown
Returns your levelnoneeveryonenone

M​

music​

DescriptionArgumentClearanceCooldown
Creates a new music channel or sets
the current channel as music		!channel | @categoryadminnone

N​

news​

DescriptionArgumentClearanceCooldown
Replies with the latest news from category!category @number_of_articleseveryonenone

P​

ping​

DescriptionArgumentClearanceCooldown
Pings Portal and returns round trip latencynoneeveryonenone

poll​

DescriptionArgumentClearanceCooldown
Replies with a message poll, that people can vote until owner stops it!poll_question | [JSON array]everyonenone

Creates a new portal channel or sets
the current channel as portal		!channel | @categoryadminnone

roll​

run​

Runs the given command string and returns its output!exec_commandeveryonenone

set_ranks​

DescriptionArgumentClearanceCooldown

spam_rules​

DescriptionArgumentAccessibleCooldown

DescriptionArgumentClearanceCooldown

url​
