### Description

The portal locale refers to the language the current portal is set to operate. Remember that voice channels inherit the portal locale.

* Default: `en`
* Clearance: `portal`

### Options

You can set the locale to be

* __gr__ _(Greece)_
* __en__ _(English)_
* __de__ _(German)_
