### Description

_returns the input decapitalised, the first letter lowercase and the rest capitalcase_

### Input Type

string, string array

### Output Type

string

### Example

|           input           |   output    |
| :-----------------------: | :---------: |
| __IOANNIS\|decapitalise__ | __iOANNIS__ |
